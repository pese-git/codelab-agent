# Спецификация: Agent Lifecycle Events (Дельта)

## ИЗМЕНЁННЫЕ Требования

### Требование: Мультимодальный контекст ExecutionEngine

Система ДОЛЖНА передавать мультимодальное содержимое через `AgentContext.prompt` как список dict частей содержимого, позволяя стратегиям получать содержимое изображений и встроенных ресурсов.

#### Сценарий: ExecutionEngine строит контекст с мультимодальным содержимым
- **КОГДА** вызван `ExecutionEngine.build_context()` с сессией, содержащей мультимодальный промпт
- **ТОГДА** `AgentContext.prompt` содержит список dict, представляющих части содержимого (например, `[{"type": "text", "text": "..."}, {"type": "image", "data": "...", "mime_type": "..."}]`)

#### Сценарий: ExecutionEngine строит контекст с текстовым содержимым
- **КОГДА** вызван `ExecutionEngine.build_context()` с текстовым промптом
- **ТОГДА** `AgentContext.prompt` содержит `[{"type": "text", "text": "..."}]`

#### Сценарий: Обратная совместимость с текстовым промптом
- **КОГДА** вызван `ExecutionEngine.build_context()` с простой строкой промпта
- **ТОГДА** `AgentContext.prompt` по умолчанию равен `[{"type": "text", "text": prompt}]`
