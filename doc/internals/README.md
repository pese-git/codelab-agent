# Внутренние документы CodeLab

Документация для разработчиков проекта, не предназначенная для публикации на сайте.

## Структура

| Директория | Описание |
|------------|----------|
| [`architecture/`](architecture/) | Архитектура, ADR, карта проекта |
| [`roadmap/`](roadmap/) | Планы развития кодовой базы (фазы: security → architecture → quality) |
| [`archive/`](archive/) | Исторические документы, выполненные планы |

## Связанные разделы

- [Продуктовая документация](../product/) — для пользователей и разработчиков
- [Протоколы](../protocols/) — референсные спецификации (ACP, A2A, MCP)
